doicms by doidata

Web-CMS to store DOI metadata in an SQL database and create valid XML registration files.

The "Digital Object Identifier" System is an international standard to add metadata to works. There a many data fields for each DOI record, which have to be assembled into a valid XML file for upload to the Registration Agency. In order to create, view, filter and sort the DOI Codes for an online literature magazine, we prorammed this Web-CMS. It serves the following separate functions: 1) Enter DOI Metadata for a specific web document (ADMIN Mode) 2) Create a valid XML file for deployment to the RA (ADMIN Mode) 3) Make the Database of DOI Codes and Metadata browsable for normal Users 4) Offer a REST API to harvest our DOI Metadata automatically

Dependencies

Cake PHP

Install

Copy all files to your web server (PHP Code). Do not forget to change ADMIN Username/Password in the file app/config/core.php Create an SQL Database structure which fits the table and field names in the PHP application. This SQL Definition is hosted in another GIT repo named "doisql".
